필요한 도움의 종류에 가장 잘 맞는 채널을 사용해 주세요.

## 질문 및 사용법 안내

- 설정 도움말, API 질문, 디자인 피드백은 [GitHub Discussions](https://github.com/fluojs/fluo/discussions)에서 토론을 시작해 주세요.

## 버그 제보

- 재현 가능한 문제를 공유할 수 있는 경우 Bug Report 양식으로 GitHub 이슈를 열어주세요.

## 문서 개선

- 가이드가 누락되었거나, 오래되었거나, 불분명한 경우 Documentation Issue 양식으로 GitHub 이슈를 열어주세요.

## 보안 보고

- 공개 이슈나 토론에 취약점을 보고하지 마세요.
- 비공개 보고 지침은 [SECURITY.ko.md](./SECURITY.ko.md)를 따르세요.
