# support

Use the channel that best matches the kind of help you need.

## questions and usage help

- Start a discussion in [GitHub Discussions](https://github.com/fluojs/fluo/discussions) for setup help, API questions, and design feedback.

## bug reports

- Open a GitHub issue with the Bug Report form when you can share a reproducible problem.

## documentation improvements

- Open a GitHub issue with the Documentation Issue form when a guide is missing, outdated, or unclear.

## security reports

- Do not report vulnerabilities in public issues or discussions.
- Follow [SECURITY.md](./SECURITY.md) for private reporting instructions.
