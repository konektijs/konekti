- 문서가 출하된 동작을 설명하면 이곳이나 패키지 README에 둡니다.
- 문서가 향후 작업을 설명하면 GitHub Issue에 둡니다.
- 한 패키지가 소유한 주제는 이곳에 중복하지 말고 해당 패키지 README를 우선합니다.
