# toolchain contract matrix

<p><strong><kbd>English</kbd></strong> <a href="./toolchain-contract-matrix.ko.md"><kbd>한국어</kbd></a></p>

This matrix defines the public toolchain contract for applications generated by the Konekti CLI and for official examples. Use this as a reference for version pinning and build configuration.

## generated app baseline

| surface | contract | version / notes |
| --- | --- | --- |
| **TypeScript** | `v5.8+` | `strict: true`, `experimentalDecorators: false`, `module: esnext` |
| **Babel** | `v7.26+` | `@babel/plugin-proposal-decorators` with `{ version: '2023-11' }` |
| **Vite** | `v6.2+` | Used for dev bundling and build orchestration. |
| **Vitest** | `v3.0+` | Standard test runner for unit and E2E tests. |
| **Node.js** | `v22+` | Target runtime for Node-based adapters. |

## CLI & scaffolding contracts

| goal | command | output contract |
| --- | --- | --- |
| **Project Creation** | `konekti new` | Generates a standard folder structure with Fastify and Node.js. |
| **Resource Generation** | `konekti g <type>` | Produces consistent naming suffixes (`.service.ts`, `.controller.ts`). |
| **Diagnostics** | `konekti inspect` | Exports runtime graph and timing data in JSON format. |

## naming conventions (CLI output)

| type | suffix | example |
| --- | --- | --- |
| **Controller** | `.controller.ts` | `users.controller.ts` |
| **Service** | `.service.ts` | `users.service.ts` |
| **Repository** | `.repo.ts` | `users.repo.ts` |
| **DTO (Input)** | `.request.dto.ts` | `create-user.request.dto.ts` |
| **DTO (Output)** | `.response.dto.ts` | `user.response.dto.ts` |

## build configuration

Konekti-generated applications use a specialized build pipeline to handle TC39-standard decorators correctly.

1.  **Transform**: Babel applies the Stage 3 decorator transform.
2.  **Bundle**: Vite bundles the application for the target runtime.
3.  **Validate**: Vitest runs conformance tests using the same decorator setup.

Replacing these tools (e.g., using `esbuild` directly) is currently unsupported as it may bypass the necessary decorator transformations.
